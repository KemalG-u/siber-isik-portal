# Siber Işık Portal

Ruhsal dönüşüm ve meditasyon platformu.

## Özellikler
- 3D Uzay arka planı (Three.js)
- İnteraktif meditasyon seansları
- GSAP animasyonlar
- Hipnotik kullanıcı deneyimi

## Kurulum
1. Projeyi klonlayın
2. `index.html` dosyasını tarayıcıda açın

## Teknolojiler
- HTML5, CSS3, JavaScript
- GSAP
- Three.js
- Canvas Confetti
